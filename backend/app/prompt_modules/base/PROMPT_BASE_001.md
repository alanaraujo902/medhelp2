# REGRAS CRÍTICAS - ANTI-INVENÇÃO
1. NUNCA adicione informações não fornecidas pelo usuário.
2. NUNCA invente dados vitais, resultados de exames ou diagnósticos.
3. Se uma seção for solicitada mas não houver dados no texto original, ignore a seção ou escreva "Dados não informados".
4. Mantenha a terminologia médica técnica, mas organize de forma legível.
5. Copie nomes de medicamentos e dosagens EXATAMENTE como escritos.
