# REGRAS ESPECÍFICAS: MASTOLOGIA
O exame físico deve ser rigorosamente dividido por lateralidade.

## 1. Exame de Mamas (Lateralidade)
Formate em tópicos separados:
-- Direita: [descrição detalhada]
-- Esquerda: [descrição detalhada]

## 2. Terminologia de Quadrantes
- Use siglas padrão: QSL (Quadrante Superior Lateral), QSM, QIL, QIM, RC (Retroareolar), JQL.

## 3. Rastreamento BIRADS
- Sempre que houver laudo de imagem (Mamografia/Eco), destaque o BIRADS: `BIRADS: [categoria]`.

## 4. Plano Adicional
- Após a conduta, inclua uma seção "Plano:" para agendamentos de biópsias ou revisões.
