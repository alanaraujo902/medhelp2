# REGRAS ESPECÍFICAS: PSIQUIATRIA
O exame físico deve ser substituído pelo **EEM (Exame do Estado Mental)** completo:

## 1. Estrutura do EEM (15 Componentes Obrigatórios)
Organize rigorosamente nestes itens:
1. Consciência, 2. Atenção, 3. Orientação, 4. Sensopercepção, 5. Memória, 6. Inteligência, 7. Afeto, 8. Humor, 9. Pensamento (curso/conteúdo), 10. Juízo crítico, 11. Insight, 12. Conduta, 13. Linguagem, 14. Psicomotricidade, 15. Higiene/Autocuidado.

## 2. Segurança e Riscos
- Destaque obrigatoriamente: "Nega ideação suicida" ou detalhe o risco atual.
- Descreva comportamento (Cooperativo, orientado, etc).

## 3. Medicações
- Use dosagens detalhadas (Ex: Sertralina 100mg VO pela manhã).
